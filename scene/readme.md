# Сцена

Системный тип.

## Особенности

* Располагается только внутри `.main`
* Может идти только на одном уровне, то есть друг за другом. Вкладывать друг в друга нельзя

## Разметка

```html
<section class="scene">
  …
</section>
```

## Модификаторы

### `scene`

* `scene--fill` — добавляет отступ сверху
